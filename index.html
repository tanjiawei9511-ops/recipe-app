🔥 炒菜基本功

  • 热锅凉油：锅烧热再放油，油热即下菜，可防粘锅
  • 大火快炒：蔬菜类保持脆嫩，损失营养少
  • 油温判断：油面起纹时为准，筷子入油冒泡即可
  • 后放盐：盐放太早出水多，且碘易挥发
大厨秘诀：炒菜前将食材沥干水分，避免油爆

🥩 肉类烹饪技巧

  • 逆纹切肉：切断肌肉纤维，肉更嫩
  • 腌制上劲：加料酒、盐、淀粉抓匀，锁住水分
  • 冷水下锅：炖肉时冷水下锅，蛋白质慢慢变性更嫩
  • 炒肉嫩滑：滑油或滑水，断生即出锅
  • 去腥三件套：料酒、姜、葱是去腥标配
大厨秘诀：炒肉片时裹蛋清淀粉，嫩度翻倍

🥚 蛋类烹饪技巧

  • 蒸蛋羹：蛋液与水1:1.5，温水下锅，小火蒸
  • 炒鸡蛋：加几滴水更蓬松，热油下锅
  • 荷包蛋：水微开时磕入，关火焖熟
  • 皮蛋：选斑点分布均匀的，切时涂蜡油不粘刀
大厨秘诀：蒸蛋时盖盘子或保鲜膜，防止水滴

🍚 米饭烹饪技巧

  • 淘米：轻柔洗1-2次，保留营养
  • 浸泡：冷水泡30分钟，米饭更饱满
  • 水量：米:水=1:1.2，新米少放水
  • 焖饭：跳闸后再焖5分钟，口感最佳
大厨秘诀：加几滴香油或白醋，米饭更香亮

🥬 蔬菜烹饪技巧

  • 焯水：加盐和几滴油，保持翠绿
  • 快炒：旺火快炒1-2分钟，锁住营养
  • 盐渍：丝瓜等切后盐渍5分钟，去多余水分
  • 配色：木耳、胡萝卜、红椒可以丰富色彩
大厨秘诀：蔬菜中的维生素C怕高温，能生吃的建议凉拌

🍜 面食烹饪技巧

  • 煮面：水开下面，加凉水反复，劲道有弹性
  • 防粘：加盐或几滴油，防止面条粘连
  • 饺子：水开下饺子，点三次冷水
  • 蒸馒头：冷水上锅，二次醒发更松软
大厨秘诀：和面时加少许盐或鸡蛋，面食更筋道

🧂 调味黄金法则

  • 糖：提鲜增甜，放早了会焦化上色
  • 醋：去腥解腻，出锅前放更香
  • 酱油：生抽提鲜，老抽上色
  • 料酒：去腥提香，高温烹调时使用
  • 花椒：炒香后捞出，只留麻味
  • 鸡精味精：出锅前放，高温会失去鲜味
大厨秘诀：调味比例——咸鲜味：盐1:酱油2；糖醋味：糖3:醋2:酱油1

🍲 汤类烹饪技巧

  • 冷水下料：肉类冷水下锅，撇浮沫去腥
  • 大火催白：奶白汤需要大火持续沸腾
  • 小火清汤：清汤类宜小火慢炖
  • 盐后放：汤出锅前放盐，肉质更嫩
大厨秘诀：炖骨头汤时加几滴醋，钙质更容易析出

麻婆豆腐｜川菜

收藏
食材
  • 嫩豆腐300g（切2cm见方块）
  • 猪肉末100g（肥瘦3:7最佳）
  • 郫县豆瓣酱2大勺、花椒粉1小勺、蒜苗1根
  • 姜蒜末、生抽、淀粉、盐
做法
  1. 豆腐切小块，开水加少许盐，焯烫2分钟去豆腥，捞出沥干
  2. 锅中放少许油，中小火炒香猪肉末至变色出油
  3. 加入郫县豆瓣酱炒出红油，放姜蒜末爆香
  4. 加清水或高汤300ml，大火烧开
  5. 放入豆腐，轻轻推匀，小火煮3-5分钟
  6. 加少许生抽调味，水淀粉勾芡至汤汁浓稠
  7. 关火后撒花椒粉和蒜苗末，滴几滴香油即可
技巧：用嫩豆腐更滑嫩；花椒粉出锅前撒更香更麻

回锅肉｜川菜

收藏
食材
  • 五花肉300g（带皮）
  • 青蒜苗2根（斜切段）
  • 郫县豆瓣酱1.5大勺、甜面酱1小勺、姜蒜片
  • 料酒、白糖少许
做法
  1. 五花肉冷水下锅，加姜片和料酒，煮至八成熟（约20分钟）
  2. 捞出放凉，切2-3mm厚的薄片（稍冷冻更好切）
  3. 锅烧热不放油，直接下肉片煸炒至微卷出油
  4. 盛出多余油脂，留少许底油
  5. 下姜蒜片爆香，加豆瓣酱炒出红油
  6. 加甜面酱和少许白糖调味
  7. 放入蒜苗段，大火快速翻炒30秒出锅
技巧：肉片稍冷冻更好切；蒜苗别炒太久保持脆嫩

鱼香肉丝｜川菜

收藏
食材
  • 猪肉丝200g（用料酒、淀粉、盐腌制10分钟）
  • 胡萝卜丝50g、木耳丝50g、青椒丝30g
  • 泡椒30g（切细）、姜蒜末、葱花
鱼香汁配方

生抽1勺、陈醋1勺、白糖1勺、淀粉1勺、清水3勺、少许盐

做法
  1. 锅中倒油，油热后滑炒肉丝至变色，盛出备用
  2. 锅中留底油，炒香泡椒末和姜蒜末
  3. 加入胡萝卜丝、木耳丝、青椒丝炒软
  4. 倒回炒好的肉丝翻炒均匀
  5. 淋入调好的鱼香汁，大火快速翻炒至收汁
  6. 撒葱花出锅
技巧：鱼香汁的酸甜比例要掌握好，泡椒是关键

宫保鸡丁｜川菜

收藏
食材

  • 干辣椒6-8个（剪段）、花椒1小勺、葱段
  • 生抽、料酒、白糖、醋、淀粉
调味汁

生抽1勺、料酒1勺、白糖1勺、醋0.5勺、淀粉1勺、水2勺

做法
  1. 锅中倒油，油热后滑炒鸡丁至变色，盛出备用
  2. 锅中留底油，小火炒香干辣椒和花椒
  3. 放入葱段爆香
  4. 倒回鸡丁翻炒均匀
  5. 淋入调味汁，翻炒至汤汁浓稠
  6. 最后加入花生米快速翻匀出锅
技巧：花生米最后放保持酥脆，火候要掌握好避免花椒炒糊

水煮鱼｜川菜

收藏
食材
  • 草鱼1条（约1kg）
  • 豆芽、莴笋片
  • 干辣椒50g、花椒20g、豆瓣酱
  • 蒜末、姜末、料酒、淀粉
做法
  1. 鱼处理干净，片成薄片，用料酒、盐、淀粉腌制15分钟
  2. 鱼骨和鱼头煎香，加水熬成鱼汤
  3. 豆芽和莴笋焯熟铺在碗底
  4. 锅中炒香豆瓣酱、干辣椒、花椒，加鱼汤煮开
  5. 放入鱼片煮至变色，倒入碗中
  6. 热油浇在干辣椒和花椒上激香
技巧：鱼片要切薄，腌制时加蛋清更嫩滑

毛血旺｜川菜

收藏
食材
  • 鸭血300g、毛肚200g
  • 牛百叶、黄喉、午餐肉
  • 豆芽、藕片、土豆片
  • 干辣椒、花椒、豆瓣酱、火锅底料
做法
  1. 鸭血切块，毛肚、百叶切丝
  2. 锅中炒香豆瓣酱和火锅底料
  3. 加水烧开，先煮耐煮的食材（藕片、土豆）
  4. 再放鸭血、毛肚等烫熟
  5. 盛入碗中，铺干辣椒和花椒
  6. 热油浇上，撒香菜即可
技巧：食材不要煮太久，保持脆嫩口感

口水鸡｜川菜

收藏
食材
  • 三黄鸡1只
  • 花生碎、芝麻、葱姜蒜
  • 辣椒油、花椒粉、生抽、醋
  • 白糖、鸡汤
做法
  1. 鸡处理干净，冷水下锅煮15分钟
  2. 关火焖10分钟，放入冰水冷却
  3. 调制红油调料：辣椒油+花椒粉+生抽+醋+糖+蒜末
  4. 鸡切块装盘，淋上红油调料
  5. 撒花生碎、芝麻、葱花
技巧：鸡要冰水过凉，皮才脆肉才嫩

白切鸡｜粤菜

收藏
食材
  • 三黄鸡1只（约1kg，嫩鸡最佳）
  • 姜片10g、葱段15g、沙姜30g
  • 生抽、花生油、盐
姜葱蘸料

姜末+葱花+少许盐+热花生油+生抽调成蘸汁

做法
  1. 鸡处理干净，去除内脏和杂质，洗净沥干
  2. 锅中加足量清水，放入姜片和葱段
  3. 水烧至微沸（锅底冒小泡），放入整鸡
  4. 保持小火微沸状态，浸煮15-20分钟（根据鸡大小调整）
  5. 关火，盖上锅盖焖10分钟
  6. 捞出迅速放入冰水或冷水中浸泡至完全冷却
  7. 沥干水分，抹一层薄油，斩件装盘
  8. 搭配姜葱蘸料或沙姜酱油食用
技巧：冰水让皮脆肉嫩；火候别太大不然肉老；浸煮时可将鸡提起几次让内部受热均匀

清蒸鲈鱼｜粤菜

收藏
食材
  • 鲈鱼1条（约600-800g）
  • 姜片15g、葱段20g、红椒丝少许
  • 蒸鱼豉油2大勺、料酒1小勺、花生油
做法
  1. 鲈鱼处理干净，去鳞去鳃去内脏，洗净
  2. 鱼身两面各划2-3刀，用少许盐和料酒涂抹鱼身内外，腌制10分钟
  3. 蒸鱼盘垫上葱段，鱼身铺上姜丝
  4. 蒸锅加水大火烧开，放入鱼盘
  5. 水开上汽后蒸8-10分钟（根据鱼大小调整）
  6. 取出鱼盘，倒掉盘中的腥水
  7. 去除鱼身上的姜丝，重新铺上新鲜葱丝和红椒丝
  8. 淋上蒸鱼豉油，浇上滚烫的热油激香
技巧：时间别太长否则肉老；最后热油激香是关键，能激发葱香

蚝油生菜｜粤菜

收藏
食材
  • 生菜400g（球生菜或罗马生菜）
  • 蒜末15g、蚝油2大勺、生抽1小勺
  • 白糖少许、淀粉、香油
做法
  1. 生菜洗净，撕成大片或整棵对半切开
  2. 锅中加水烧开，加少许盐和几滴油，放入生菜焯烫10-15秒
  3. 迅速捞出，沥干水分，整齐摆入盘中
  4. 锅中放少许油，爆香蒜末
  5. 加入蚝油、生抽和少许白糖，加少量清水搅匀
  6. 水淀粉勾薄芡，滴几滴香油
  7. 将酱汁均匀浇在生菜上即可
技巧：生菜焯水时间要短，保持脆嫩；蚝油本身有咸味，生抽别放太多

叉烧肉｜粤菜

收藏
食材
  • 猪里脊肉500g（或梅花肉）
  • 叉烧酱3大勺、生抽1大勺、料酒1大勺
  • 蜂蜜2大勺、蒜末10g、姜片10g
做法
